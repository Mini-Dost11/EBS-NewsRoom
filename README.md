# EBS 뉴스 보도정보시스템 프로토타입

EBS 뉴스 제작 조직을 위한 통합 보도정보시스템 화면 프로토타입입니다. 빌드 도구 없이 정적 파일로 구성되어 있어 브라우저에서 바로 확인할 수 있습니다.

## 실행

```bash
python3 -m http.server 5173
```

브라우저에서 `http://127.0.0.1:5173`으로 접속합니다.

## 화면 구성

- 통합 뉴스룸 대시보드: 연합뉴스 수신 뉴스, 제작 현황, 오늘 런다운, 송출 준비 상태
- 기사 작성·승인: 구조화된 방송 원고, 연합뉴스 원문 연결, 데스크 체크, AI 보조 제안
- 런다운·큐시트·송출 연계: 방송 순서, 프롬프터/자막/영상 상태, 송출 전 체크리스트
